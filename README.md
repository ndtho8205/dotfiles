# dotfiles

## Requirements

Set `zsh` as your login shell:

```shell
chsh -s $(which zsh)
```

Install VIM 8.

## Install

```shell
git clone --recursive https://github.com/ndtho8205/dotfiles.git
cd dotfiles
./install.sh
```
